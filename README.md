# Task Manager App

A simple Task Manager web application built with React.

## Features
- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as complete

## Tech Stack
- React
- JavaScript
- CSS

## How to Run the Project

1. Clone the repository:
   git clone https://github.com/Anushka-Singh02102002/task-manager.git

2. Go into the project folder:
   cd task-manager

3. Install dependencies:
   npm install

4. Start the app:
   npm start

5. Open your browser at:
   http://localhost:3000
